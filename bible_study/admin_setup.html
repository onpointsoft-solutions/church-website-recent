🔧 Method 2: Fix XAMPP PHP Extensions

Enable required PHP extensions:

  1. Open XAMPP Control Panel
  2. Click "Config" next to Apache
  3. Select "PHP (php.ini)"
  4. Find and uncomment these lines (remove semicolon):
extension=mysqli extension=pdo_mysql extension=curl extension=openssl extension=mbstring
  1. Save the file and restart Apache
  2. Try accessing insert_admin.php again
